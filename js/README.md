### 선언

변수 선언 : `let`<br>
상수 선언 : `const` → 한번 할당 시 불변한다. 값이 바뀌지 않는다면 const를 사용할 것.

3. 여러 단어를 연결하면 이름 중간을 대문자로 사용한다.
4. 의미있게 작성할 것.

### 함수표현식
익명 함수 -> 이름이 없는 함수
<br>
<code>
let sum = fuction(a,b) { return a+b; }
</code><br>

익명 함수를 사용하는 이유 -> 함수를 변수에 할당하거나, 함수 자체를 매개변수로 넘긴다.<br>
코드 사이의 충돌이 생기는 것을 방지하기위해 사용한다.<br>

즉시 실행 함수 `IIFF` -> 한번만 실행하는 함수라면 함수를 정의하면서 실행할 수 있다.
<code>
(function() {명령}());<br>
(function(매개변수) {명령}(인수));
</code>

화살표 함수 -> 함수 이름이 없을 때 만 사용이 가능하다.<br>
(매개변수) -> {함수 내용}

1. 매개변수가 없을 경우<br>
` const hi = function() {
    return "안녕하세요"?;
} ` -> ` const hi = ()=>{return "안녕하세요?";} ` -> ` const hi=()->"안녕하세요?"; `

2. 매개변수가 1개인 함수<br>
` let hi = function(user) {
    document.write(user+"님, 안녕하세요?");
} `

### 이벤트와 이벤트 처리기
> 웹 브라우저나 사용자가 행하는 어떤 동작

#### 마우스 이벤트
- `click` : HTML요소를 클릭할 때
- `dbclick` : HTML요소를 더블 클릭 할 때
- `mousedown` : 사용자가 요소 위에서 마우스 버튼을 누를 때
- `mouseover` : 포인터가 요소 위로 옮겨질 때
- `mousemove` : 마우스 포인터를 움직일 때
- `mouseout` : 요소를 벗어날 때
- `mouseup` : 마우스 버튼에서 손을 뗄 때

#### 키보드 이벤트
- `keydown` : 사용자가 키를 누르는 동안
- `keypress` : 사용자가 키를 눌렀을 때
- `keyup` : 사용자가 키에서 손을 뗄 때

#### 문서 로딩 이벤트
- `abort` : 문서가 완전히 로딩되기 전에 불러오기를 멈첬을 때 이벤트가 발생합니다.
- `error` : 문서가 정확히 로딩되지 않을 때 이벤트가 발생합니다.
- `load` : 문서 로딩이 끝나면 이벤트가 발생합니다.
- `resize` : 문서 화면의 크기가 바뀌었을 때 이벤트가 발생합니다.
- `scroll` : 문서 화면이 스크롤 되었을 때 이벤트가 발생합니다.
- `unload` : 문서 화면에서 벗어날 때 이벤트가 발생합니다.

#### 폼 이벤트
- `blur` :  폼 요소에서 포커스가 벗어났을 때
- `change` : 목록이나 체크 상태 등이 변경되면 이벤트가 발생합니다.
- `focus` : 폼 요소에 포커스가 놓였을 때 이벤트가 발생합니다.
- `reset` : 폼이 리셋 되었을 때 이벤트가 발생합니다.
- `submit` : submit 버튼을 클릭했을 때 이벤트가 발생합니다.

### DOM 요소에 접근하기
1. `document.getElementById("아이디 명") -> # 생략`<br>
2. `document.getElementByClassName("클래스 명") -> . 생략`<br>
3. `document.getElementByTagName("태그 명")`<br>

#### 하나만 콕 집어내는 querySelector() 함수<br>
- `document.querySelector(선택자) -> # , . 붙여줘야함`

#### 모든 요소를 가져오는 querySelectorAll() 함수<br>
- `document.querySelectorAll(선택자) -> # , . 붙여줘야함`
> 같은 선택자를 사용하는 개체들이라면 NodeList()로 가져올 수 있다. <br>
배열처럼 인덱스 값을 이요하여 접근한다.
